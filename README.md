# 🏥 Hospital Management System (DSA)

A comprehensive, menu-driven **C++ Console Application** designed to streamline healthcare data management. This project demonstrates the practical application of core **Data Structures and Algorithms (DSA)** to handle patient records, appointment scheduling, and emergency queues efficiently.

---

## 🚀 Key Features

- **Patient Registration & Records:** Efficiently add, search, update, and display patient information.
- **Appointment Scheduling:** Manage doctor appointments seamlessly using structured data.
- **Emergency Priority Queue:** Handle critical patients based on urgency using advanced queue mechanics.
- **Discharge Management:** Track and process discharged patients while maintaining active logs.

---

## 🛠️ Data Structures Used

This project highlights the implementation of fundamental computer science concepts:
1. **Linked Lists:** Used for dynamic memory allocation and managing active patient records where insertions and deletions are frequent.
2. **Stacks:** Employed to track recent transactions, undo actions, or maintain historical logs of discharged patients (Last-In, First-Out).
3. **Queues:** Used for managing regular and emergency patient check-up lines (First-In, First-Out).

---

## 💻 Tech Stack & Tools

- **Language:** C++ (Standard Template Library & Custom Structural Blueprints)
- **Environment:** Console/Terminal Based Application
- **Concepts:** Object-Oriented Programming (OOP) & Data Structures (DSA)

---

## 📖 How to Run the Project

1. Clone this repository or download the source files.
2. Open the project in any C++ IDE (like VS Code, Dev-C++, or Code::Blocks).
3. Compile the `main.cpp` file using a GCC compiler.
4. Run the executable file to interact with the system through the terminal menu.
